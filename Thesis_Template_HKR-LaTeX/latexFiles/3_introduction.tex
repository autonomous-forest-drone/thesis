\subsection{First Subsection}\label{section:IntroductionSubsection}
\noindent
Subsection. You can use enumerate:
\begin{enumerate}
\item First
\item Second
\end{enumerate}

\noindent
Specify the values, I, II, III, ...:
\begin{enumerate}[I.]
\item First
\item Second
\end{enumerate}

\noindent
Specify the values Q1, Q2, ...:
\begin{enumerate}[Q1.]
\item First
\item Second
\end{enumerate}

\noindent
Specify the values A, B, C...:
\begin{enumerate}[A.]
\item First
\item Second
\end{enumerate}

\noindent
Or you can use itemize:
\begin{itemize}
    \item First
    \item Second
\end{itemize}\par\vspace{3mm}

\newpage %I don't want to have the text on one page and the itemize on the next, so I create new page!

\noindent
You can choose what you want to use instead of the dot in itemize, e.g. a right arrow.
\begin{itemize}[$\rightarrow$]
    \item First
    \item Second
\end{itemize}\par\vspace{3mm}
